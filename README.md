## Getting Started

To run the project locally, follow these steps:

1. **Clone the repository**
    ```bash
    git clone https://github.com/Hasan-Uddin/flutter_onboard_screen_alarm_app.git
    cd flutter-bmi-calculator
3. **Install dependencies**
    ```bash
    flutter pub get
4. **Run the app**
    ```bash
    flutter run
5. **build**
    - for web
        ```bash
        flutter build web
    - for Android
        ```bash
        flutter build apk

and so on...
